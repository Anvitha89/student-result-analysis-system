# Student Result Analysis System

## Project Overview

This project analyzes student academic performance using Python and Pandas. It reads student data from Excel files, performs data cleaning, calculates totals, percentages, grades, merges student details, and generates final reports in Excel and CSV formats.

## Technologies Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

## Features

* Load data from Excel files
* Calculate Total Marks
* Calculate Percentage
* Assign Grades
* Perform Data Cleaning
* Remove Duplicate Records
* Merge Multiple Datasets
* Export Final Reports to Excel and CSV

## Dataset Information

The project uses the following datasets:

* Student_Marks
* Student_Marks_Dirty
* Student_Details

## Grade Criteria

* Percentage < 40 → Fail
* 40 - 60 → Pass
* 60 - 80 → First Class
* Above 80 → Distinction

## Output Files

* Final_Student_Report.xlsx
* Final_Student_Report.csv
